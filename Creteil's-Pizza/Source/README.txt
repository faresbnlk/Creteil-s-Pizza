Bonjour, le fichier pizza.sql regroupe l'ensemble des tables
tandis que 2017_projet4_pizza.sql est une base de données avec l'ensemble des tables.
Étant donné que sur le site https://pw.lacl.fr/gbd, les étudiants n'ont pas le droit de créer
une nouvelle base de données mais seulement de nouvelles tables,
je me suis permis de créer un ensemble de tables sans une base de données.

Donc si vous voulez utiliser https://pw.lacl.fr/gbd, importez plutôt le fichier pizza.sql.
